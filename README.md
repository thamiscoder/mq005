# Atv Media Query 05

Projeto de página web estática desenvolvido como estudo do curso **HTML5 e CSS3 do Curso em Vídeo**, no **Módulo 04, Capítulo 25 - Menu Responsivo com Media Queries**.

O objetivo deste exercício é criar uma página simples com menu adaptável, exibindo um menu estilo **hambúrguer** em telas menores e um menu horizontal em telas maiores.

## Sobre o projeto

O `mq005` é um estudo de responsividade aplicado à navegação de um site. A página possui cabeçalho, menu, conteúdo textual e regras CSS que mudam o comportamento do layout conforme o tamanho da tela.

Em dispositivos menores, o menu fica recolhido atrás do ícone de hambúrguer. Em telas maiores, o menu passa a ser exibido de forma horizontal, deixando a navegação mais confortável.

## Funcionalidades estudadas

- Estruturação de cabeçalho e menu com HTML5;
- Criação de menu responsivo;
- Uso de ícone/menu hambúrguer;
- Estilização de links e estados de interação;
- Uso de media query para telas a partir de 768px;
- Alteração do layout do menu conforme o tamanho da tela;
- Definição de largura fixa para o conteúdo em telas maiores;
- Organização de CSS base e CSS responsivo.

## Tecnologias utilizadas

- HTML5;
- CSS3;
- Media Queries.

## Como visualizar o projeto

1. Baixe ou clone este repositório:

```bash
git clone https://github.com/thamiscoder/mq005.git
```

2. Acesse a pasta do projeto:

```bash
cd mq005
```

3. Abra o arquivo `index.html` no navegador.

Para testar o menu responsivo, diminua e aumente a largura da tela pelo modo responsivo do navegador.

## Objetivo de aprendizado

Este projeto foi criado para praticar:

- menus responsivos;
- comportamento visual em telas pequenas e grandes;
- uso de breakpoints;
- media queries com `min-width`;
- adaptação de navegação para diferentes dispositivos;
- construção de layout simples e funcional.
